CSS and JS (Scroll down to get the links)

<link rel="stylesheet" href="modal.css"> <script src="modal.js"></script>

For modal buttons and assign any of your required ID (id="modal") name like below :

<button id="modal" class="modalBtn">Modal</button>

For modal content body make sure you give the 'data-modal' attribute name same as the button id (data-modal="#modal") and a classname "modal" as shown below :

<div data-modal="#modal" class="modal">...

Make sure you also have this close button with the class name of "closeBtn" :

<div data-modal="#modal" class="modal"> <div class="modal-content"> <div class="modal-header"><span>title here</span> <span class="closeBtn">close button</span> </div>...

Full HTML syntax given below

<button id="modal" class="modalBtn">Modal</button> <div data-modal="#modal" class="modal"> <div class="modal-content"> <div class="modal-header"><span><strong>Title here</strong></span> <span class="closeBtn">close button (use your own icon)</span> </div> <div class="modal-body"><p>Modal body here</p> </div> <div class="modal-footer"><p>footer here</p></div> </div> </div>
