For alert content body make sure you set the 'data-alert' attribute and a class name of "tm-alert" to get the custom styling

<div data-alert="#tm-alert" class="tm-alert">...

Make sure you also have this close button with the class name of "closeBtn" :

<div data-alert="#tm-alert" class="tm-alert"> <div class="tm-alert-content"> <div class="tm-alert-header"><span><strong>This is an alert title</strong></span> <span class="closeBtn">CLOSE BUTTON</span> </div>

Full HTML syntax given below

<div class="wrapper"> <div data-alert="#tm-alert" class="tm-alert"> <div class="tm-alert-content"> <div class="tm-alert-header"><span><strong>This is an alert title</strong></span> <span class="closeBtn">&times;</span> </div> <div class="tm-alert-body"><p>This is alert body</p> </div> <div class="tm-alert-footer"><p>This is alert footer</p></div> </div> </div> </div>

If you want to set a custom timeout(Not mandatory) to auto hide the alert just call this function of "timoutAlertInSeconds(seconds) which take an parameter of seconds"

<script> timeoutAlertInSeconds(10); </script>
